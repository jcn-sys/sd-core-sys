\# スマホドクター 店舗管理・公開システム (SD-CORE)



\## 概要

Google Apps Script (GAS) とスプレッドシートを基盤とした、店舗情報管理システムです。

店舗側からの修正申請、本部による承認フロー、および一般ユーザー向けの一覧・詳細表示機能を備えています。



\## 主な機能

\- \*\*店舗管理画面\*\*: 各店舗が自身の情報（画像、サービス、SNS等）を編集・申請。

\- \*\*承認システム\*\*: 本部がスプレッドシート上で内容を確認し、ワンクリックで本番反映。

\- \*\*動的ショップページ\*\*: 承認済みデータを元に、レスポンシブな店舗ページを自動生成。

\- \*\*画像最適化\*\*: Googleドライブ保存した画像を高速表示用に変換。



\## 技術スタック

\- Google Apps Script (JavaScript)

\- HTML5 / CSS3 (Flexbox, CSS Variables)

\- Google Spreadsheet / Google Drive API
